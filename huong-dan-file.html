Hướng dẫn xuất file in

Để đảm bảo chất lượng hình in tốt nhất, vui lòng tuân thủ các quy định sau

1. Định dạng file

Chúng tôi chấp nhận các định dạng file sau:

  • .PNG (Khuyên dùng): Nền trong suốt (Transparent Background).
  • .PDF, .AI, .PSD: File gốc thiết kế, đã convert font thành vector (outline/shape).

2. Độ phân giải (Resolution)

File in phải đạt tối thiểu 300 DPI (Dots Per Inch) ở kích thước in thực tế.

Ví dụ: Hình in ngực trái 10cm x 10cm cần file kích thước 10cm x 10cm với 300dpi.

3. Hệ màu (Color Mode)

Sử dụng hệ màu CMYK để màu sắc khi in ra chuẩn nhất so với thiết kế.

Lưu ý: File hệ màu RGB khi in ra thường bị xỉn màu hơn so với nhìn trên màn hình điện thoại/máy tính.

4. Nét mảnh và Độ trong suốt

  • Không sử dụng hiệu ứng bóng mờ (Drop Shadow) hoặc độ trong suốt (Coverage) dưới 30% vì máy in sẽ không phun được lót trắng, dẫn đến hình bị loang lổ.
  • Đường nét nhỏ nhất (Stroke weight) tối thiểu 1pt (0.35mm) để đảm bảo độ bám dính.
